### How to run

dev:

from root:

```bash
    go run apps/backend/cmd/main.go
```

from apps/backend:

```bash
    go run cmd/main.go
``` 

or with live reload (air):

```bash
    air -c .air.toml
```

### Fresh   

Fresh is a tool that allows you to run your application with hot reload.

To install:

```bash
    go install github.com/air-verse/air@latest
```
