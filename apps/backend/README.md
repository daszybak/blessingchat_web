### How to run

dev:
    from root:
    ``` 
        go run apps/backend/main.go
    ```
    from apps/backend:
    ``` 
        go run main.go
    ``` 
    or with hot reload (fresh):
    ``` 
        fresh
    ```

### Fresh   

Fresh is a tool that allows you to run your application with hot reload.

To install:
``` 
    go install github.com/gravityblast/fresh@latest
```
